TI-(very)-BASIC

This page should get you up to speed about how to use your calculator and the simplest commands

How To Get to Programs

  • Programs can be accessed by pressing the PRGM button on your calculator
  • At the EXEC tab, pressing enter over a program will run the program
  • The EDIT tab lets you modify existing programs
  • The NEW tab allows you to create a new program from scratch.
  • Both the EDIT and NEW tabs will take you to the calculator's built-in program editor

How to Actually Write Programs and Simple Commands

  • While in the program editor, pressing PRGM does not return you to the default program list. Instead it takes you to a list of programming commands
  • To store a variable, type a value and then press "STO" followed by "ALPHA" and a key with a number on top. This stores whatever value to the letter variable you gave it
    • For example "100 + 44→A" would store 144 to the A variable.
    • These are only float variables. There are other places to store things such as lists and strings but those can be covered later
  • The display or print command on your calculator is called "Disp". It can be found on the I/O tab of the programming menu at number 3.
    • For example: "Disp 1+5" Will print out 6
    • Not super important but strings and numbers show on different sides of the screen. Numbers are right-aligned and strings are left-aligned
  • To accept user input, use either the "Input" or "Prompt" command
    • The "Input" command takes two parameters, prompt string and a variable name. Typing the prompt string can be annoying but it does make it easier to remember what you need to type in
    • The "Prompt" command makes this easier by just accepting a variable as a parameter. Instead of you picking the prompt string it just shows the variable name followed by a question mark
    • "Input" and "Prompt" are numbers 1 and 2 on the I/O tab, respectively
