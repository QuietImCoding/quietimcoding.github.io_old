TI-(not-so)-BASIC

This page has slightly more advanced commands and assumes you already know the very basics shown on TI-(very)-Basic.

Other Variable Formats

  • In addition the simple letter variables, your calculator can also handle lists, matricies, and (sort of) strings
  • To save a string variable, type something in quotes, press "STO" and then press "VARS" to access the variables page. Go down to String and then select Str1 or 2 or another one.
    • It's worth noting that the support for strings on calculators is minimal. Don't expect to do any fancy string operations
  • Lists can be seen on your calculator on the number keys from 1 to 6 and are denoted with the names L1-L6. They have quite a few functions accessible by pressing "SECOND" then "STAT". Basic operation is similar to other programming languages. To get the value at a certain index type the list name and then the index in parentheses like L1(4).
    • Note: List indexes start at one
    • To extend a list use L1(dim(L1)+1)
      • dim is the length function found in the LIST menu
  • Matricies can be used the way they are used in your math classes. If you haven't used matricies on your calculator before, they are very powerful but also too complicated to cover in this tutorial

Drawing Pretty Things

  • Drawing in a program can be accomplished through the "DRAW" menu, accessed by pressing "SECOND" and then "PRGM"
  • Most drawing commands are relative to the coordinate plane.
    • For example, the Line command, which takes 4 parameters: X1, Y1, X2, and Y2. (An optional color parameter is available on some calculators)
  • The Text command is the only exception. It's coordinates are absolute and kind of weird. X and Y coordinates 1, 1 are at the top left of the screen and fractional coordinates are not allowed.
